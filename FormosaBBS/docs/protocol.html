    FILEPUT type filename title [PATH level1 lebel2 ..]

    送上一個檔案

    type 站內檔或是站外檔 （0— 站內 1— 站外）
    filename 上傳檔案名稱
    title 檔案解說

    type=1 （為站外檔時）

    收到OK 之後，再上傳檔案所在的FTP site ，以及所在的目錄。

    如：

    S>800 OK

    C>ftp.nsysu.edu.tw

    C>/upload/win95/

    800 OK

    type=0 （為站內檔時）

    收到OK 之後，便開始上傳檔案內容。

    做完動作後便斷線
