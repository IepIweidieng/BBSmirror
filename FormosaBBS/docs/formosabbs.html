FormosaBBS - Formosa Bulletin Board System


國立中山大學電算中心網路組 所開發的一套 telnet-based BBS 系統,
目前支援數種 Unix 作業系統 (如: SunOS, Solaris, Linux, FreeBSD, AIX).

  • 目前使用本系統的代表站台有:
    • FormosaBBS [bbs.nsysu.edu.tw] (OS: Solaris) Web-BBS
    • AnimeBBS [bbs.anime.nsysu.edu.tw] (OS: Linux) Web-BBS
  • 最新版本 FormosaBBS v1.1.1 完整程式可在以下網址下載
    ftp://ftp.nsysu.edu.tw/NSYSU/BBS/FormosaBBS-1.1.1.tar.gz

  • 若第一次安裝使用, 請先閱讀 QuickStart 或 INSTALL 說明文件
    • ftp://ftp.nsysu.edu.tw/NSYSU/BBS/QuickStart
    • ftp://ftp.nsysu.edu.tw/NSYSU/BBS/INSTALL

  • 想知道 FormosaBBS v1.1.1 與舊版有何不同? 請閱讀
    ftp://ftp.nsysu.edu.tw/NSYSU/BBS/NEWS.111
    ftp://ftp.nsysu.edu.tw/NSYSU/BBS/NEWS
  • 本版新支援 FreeBSD, 相關資訊, 請閱讀
    ftp://ftp.nsysu.edu.tw/NSYSU/BBS/README.FreeBSD
  • 若有疑問請先閱讀 FAQ (Frequently Asked Questions)
  • 身份認證問答集 (Ident-FAQ) [新增!!]
